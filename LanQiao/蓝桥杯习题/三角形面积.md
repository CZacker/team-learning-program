# 三角形面积

已知三角形三个顶点在直角坐标系下的坐标分别为：

(2.3, 2.5)

(6.4, 3.1)

(5.1, 7.2)

求该三角形的面积。

注意，要提交的是一个小数形式表示的浮点数。
要求精确到小数后3位，如不足3位，需要补零。


```
a=(2.3,2.5)
b=(6.4, 3.1)
c=(5.1, 7.2)
di=((a[0]-b[0])**2+(a[1]-b[1])**2)**(1/2)
k=(a[1]-b[1])/(a[0]-b[0])
b_=a[1]-k*a[0]
h=(c[1]-c[0]*k-b_)/(1+k**2)**(1/2)
print((h*di)/2)
```

    8.795000000000002
